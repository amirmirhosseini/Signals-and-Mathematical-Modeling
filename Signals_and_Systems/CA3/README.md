# Signals and Systems - Computer Assignment 3

This repository contains the MATLAB implementation and project report for the third computer assignment of the Signals and Systems course at the University of Tehran.

## Project Overview
* **Part 1 (Image Steganography):** Implementing custom coding and decoding functions to hide a 5-bit binary encoded text message inside the $6 \times 6$ pixel blocks of a grayscale image based on localized variance thresholds.
* **Part 2 & 3 (Classification & Diabetes Detection):** Building a basic data-driven classifier using patient clinical features to predict diabetes, followed by testing and evaluating model accuracy on a separate test dataset.
* **Part 4 (Signal Convolution Analysis):** Mathematical and graphical simulation of continuous-time convolutions ($a*a$, $a*b$, $b*c$, etc.) between multiple standard signal waveforms to analyze system outputs.
