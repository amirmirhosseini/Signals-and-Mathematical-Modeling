# Signals and Systems - Computer Assignment 1

This repository contains the MATLAB implementation for the first computer assignment of the Signals and Systems course at the University of Tehran.

## Project Overview
* **Part 1 (MATLAB Basics):** Introduction to fundamental operations, vector/matrix manipulations, and basic signal plotting.
* **Part 2 (Radar Signal Simulation):** Simulation of a radar system for target distance estimation. It uses cross-correlation (template matching) to calculate the distance based on signal time delay and includes a Monte Carlo analysis to evaluate the estimation error under varying noise power levels.
