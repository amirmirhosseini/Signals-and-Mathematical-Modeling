# Signals and Systems - Computer Assignment 7

This repository contains the MATLAB implementation, Simulink models, and project report for the seventh computer assignment of the Signals and Systems course at the University of Tehran, focusing on Laplace transform analysis and continuous-time system modeling.

## Project Overview
* **Part 1 (RLC Circuit Analysis):** Modeling a series RLC network in the Laplace domain, deriving its transfer function $H(s)$, and simulating system behavior under specific component values ($R, L, C$).
* **Part 2 (Vehicle Suspension System Simulation):** Analyzing an automobile suspension mechanism across various damping coefficients ($B = 0, 10, 100$). The project compares theoretical time-domain responses with dynamic Simulink environment simulations to evaluate cabin stability, overshoot, and convergence rates.
* **Part 3 (Differential Equations with Initial Conditions):** Utilizing the unilateral Laplace transform to solve second-order differential equations featuring non-zero initial conditions ($y(0^-), y'(0^-)$) using MATLAB's symbolic toolbox (`syms`, `laplace`, `ilaplace`).
