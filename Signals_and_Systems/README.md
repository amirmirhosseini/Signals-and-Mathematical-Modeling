# Signals and Systems - Computer Assignments

This repository contains the MATLAB implementations, Simulink models, and project reports for the 7 computer assignments (CAs) of the Signals and Systems course at the University of Tehran.

## Repository Structure

### CA1: Introduction to MATLAB & Transceiver Channel Simulation
* **Topics:** Vectorization, basic signal generation, distance estimation using correlation/template matching, and evaluating noise effects on estimation accuracy.

### CA2: Image Processing & License Plate Extraction
* **Topics:** Custom RGB-to-grayscale conversion, binarization, morphological filtering for noise removal, license plate boundary localization, and character segmentation.

### CA3: Image Watermarking & Continuous Convolution
* **Topics:** Steganography via embedding text messages into grayscale image blocks based on local variance, paired with continuous-time analytical and numerical convolution.

### CA4: Amplitude Modulation (AM) & Communication Systems
* **Topics:** Simulating an AM transmitter and receiver with time synchronization, and evaluating bit-rate performance against additive Gaussian noise.

### CA5: Frequency Domain Analysis & FSK Modulation
* **Topics:** Analyzing frequency resolution limits using FFT, implementing Frequency-Shift Keying (FSK) encoding/decoding, and testing noise immunity under different transmission speeds.

### CA6: Audio Signal Processing & Piano Simulation
* **Topics:** Synthetic musical note generation, melody composition, and an automated Fourier-based audio decoder to track played piano notes and isolate time boundaries.

### CA7: Laplace Transform & Automobile Suspension Modeling
* **Topics:** Solving continuous-time differential equations with non-zero initial conditions via unilateral Laplace transforms, and vehicle suspension system modeling in Simulink across various damping coefficients.
