# Engineering Probability and Statistics - Computer Assignment 0

This repository contains the implementations, simulations, and analytical reports for the introductory computer assignment (CA0) of the **Engineering Probability and Statistics** course at the **University of Tehran**. 

All computational tasks, algorithmic processing pipelines, and data analysis routines were developed from scratch using **Python**, leveraging **NumPy** for vectorized numerical computations and **Matplotlib/Seaborn** for data visualization.

---

## Project Objectives & Implementation Strategy

The primary focus of this assignment is to establish a strong foundation in high-performance vectorized operations, study voting systems using probabilistic simulations, and implement a foundational statistical classifier from scratch.

### 1. Vectorized Matrix Operations (NumPy)
* **Loop-Free Data Processing:** Designed and optimized array-manipulation routines bypassing explicit Python loops to execute fast column-wise normalization, statistical metric extraction, and mathematical indexing directly via vectorized hardware acceleration.

### 2. Probabilistic Voting System & Multi-Dimensional Simulation
* **Bayesian Condorcet Model Evaluation:** Implemented exact analytical probability models to evaluate the reliability of democratic majority voting frameworks under varying competence levels.
* **Monte Carlo Stochastic Simulation:** Developed a large-scale stochastic simulation environment to empirically model voting behavior, validating experimental convergence behavior against theoretical predictions through 3D accuracy surface mapping.

### 3. Statistical Spam Classification (Naive Bayes Classifier)
* **Bag-of-Words (BoW) Representation:** Engineered a text-processing pipeline to map raw text datasets into discrete feature frequency profiles.
* **Prior and Conditional Probability Estimation:** Built a binary Naive Bayes text classifier from scratch using conditional probabilities to distinguish between distinct message classes.
* **Numerical & Algorithmic Optimization:** Integrated Laplace Smoothing techniques to eliminate zero-probability errors for out-of-vocabulary elements, and transitioned calculations into the logarithmic domain to fully prevent floating-point underflow on long text streams.
